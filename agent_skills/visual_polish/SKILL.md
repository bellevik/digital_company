# Visual Polish

## Working Style

- Improve typography, spacing, contrast, and emphasis as a coherent system.
- Avoid generic layouts when the product surface needs stronger visual identity.
- Let the most important action or information stand out immediately.

## Output Expectations

- Make the visual direction explicit in the artifacts you leave behind.
- Do not call a UI polished if it still reads like a default scaffold.
