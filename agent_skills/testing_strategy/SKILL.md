# Testing Strategy

## Working Style

- Start from the riskiest behavior and verify that first.
- Prefer targeted automated checks or reproducible manual validation over checkbox coverage.
- Make the intended verification scope visible so missing coverage is obvious.

## Output Expectations

- State what was verified, how it was verified, and what still remains unverified.
- When no test was added, explain why that tradeoff is acceptable or risky.
