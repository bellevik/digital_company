# Studio Asset Sources

This folder contains the starter placeholder assets for the future `Studio` view.

These are intentionally placeholder-friendly and swap-friendly. The scene system should treat
them as interchangeable inputs, not as final art direction.

## Fetched Into Repo

### 1. Roguelike Indoor pack
- Source: OpenGameArt / Kenney
- URL: https://opengameart.org/content/roguelike-indoor-pack
- License: CC0
- Why: strong base for interior furniture, desks, chairs, shelves, and room dressing
- Local folder: `vendor/roguelike-indoor`

### 2. factory tileset
- Source: OpenGameArt / rubberduck with Kenney collaboration
- URL: https://opengameart.org/content/factory-tileset
- License: CC0
- Why: useful for server-room, terminal-bay, pipes, machines, and industrial cyberpunk mood
- Local folder: `vendor/factory`

### 3. Puny Characters
- Source: OpenGameArt / Shade
- URL: https://opengameart.org/content/puny-characters
- License: CC0
- Why: cute/chibi top-down base sprites with good placeholder readability
- Local folder: `vendor/puny-characters`

### 4. Hand-Drawn Square Characters Animated 8 Directions Top Down free cc0
- Source: OpenGameArt / rgsdev
- URL: https://opengameart.org/content/hand-drawn-square-characters-animated-8-directions-top-down-free-cc0
- License: CC0
- Why: animated 8-direction walking/idle base for agent movement prototyping
- Local folder: `vendor/square-characters`

### 5. Free CC0 Top Down Tileset Template Pixel Art
- Source: OpenGameArt / rgsdev
- URL: https://opengameart.org/content/free-cc0-top-down-tileset-template-pixel-art

## Intended Use

For `Studio v1`, these assets are meant to support:
- top-down spacious office floor layout
- visible task zones
- chibi agent movement
- role-coded placeholder avatars
- cyberpunk tower mood through layout, recolor, lighting overlays, and signage

The system should be built so these assets can be replaced later with:
- custom art
- commissioned art
- purchased packs
- mixed-source local-only packs
